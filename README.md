# Simple Python Calculator

A simple yet robust calculator implementation in Python with support for basic arithmetic and advanced mathematical operations.

## Features

- Basic Operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with zero division handling)

- Advanced Operations:
  - Logarithm (base 10)
  - Square
  - Square Root
